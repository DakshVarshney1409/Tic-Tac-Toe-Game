TIC TAC TOE

Rules of the game:

  • The game is played on a grid that is 3x3 squares.

  • First player starts as X and second player starts as O.

  • The first player to get 3 of his/her marks in a row(up/down/across/diagonal) is the winner

  • When all 9 squares are full, the game is over. If no player has 3 marks in a row, the game ends in a tie.
